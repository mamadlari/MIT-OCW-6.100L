# 💻 پروژه‌های درس MIT 6.100L (مجموعه تمرین‌های ۱ و ۲)

این مخزن شامل حل تمرین‌های (Problem Sets) درس مقدمه‌ای بر علوم کامپیوتر و برنامه‌نویسی با پایتون دانشگاه MIT است.

---

## 📌 مجموعه تمرین ۱: برنامه‌ریزی مالی و پس‌انداز (Problem Set 1)

این تمرین روی محاسبات مالی، مدیریت حلقه‌ها و استفاده از الگوریتم **جستجوی دودویی (Bisection Search)** تمرکز دارد.

### بخش‌ها:

* **بخش الف (Part A):** محاسبه تعداد ماه‌های مورد نیاز برای جمع‌آوری پول پیش‌پرداخت خرید خانه با فرض ثابت بودن حقوق.
* **بخش ب (Part B):** توسعه برنامه بخش اول با در نظر گرفتن افزایش حقوق شش‌ماهه (`semi_annual_raise`).
* **بخش ج (Part C):** پیدا کردن کمترین نرخ بازدهی سرمایه‌گذاری (`r`) با استفاده از **جستجوی دودویی** برای رسیدن به پیش‌پرداخت یک خانه‌ی ۸۰۰,۰۰۰ دلاری در ۳۶ ماه (با دقت ۱۰۰ دلار).

### فایل‌ها:

* `ps1a.py`: کد مربوط به پس‌انداز با حقوق ثابت
* `ps1b.py`: کد مربوط به پس‌انداز همراه با افزایش حقوق شش‌ماهه
* `ps1c.py`: کد مربوط به الگوریتم جستجوی دودویی برای پیدا کردن نرخ بهره بهینه

---

## 🎮 مجموعه تمرین ۲: بازی هنگ‌من / حدس کلمه (Problem Set 2)

این تمرین شامل پیاده‌سازی کامل بازی کلاسیک و تعاملی هنگ‌من (Hangman) در محیط ترمینال است که به صورت شیءگرا و با توابع کمکی مجزا طراحی شده است.

### ویژگی‌های کلیدی:

* **توابع کمکی:** پیاده‌سازی توابع بررسی وضعیت برنده شدن بازیکن (`has_player_won`)، نمایش کلمه حدس‌زده‌شده با ستاره (`get_word_progress`) و مدیریت حروف باقی‌مانده الفبا (`get_available_letters`).
* **مدیریت هوشمند حدس‌ها:** کسر حدس‌های باقی‌مانده بر اساس حروف صدادار (۲ حدس) و بی‌صدا (۱ حدس) در صورت حدس اشتباه. ورودی‌های تکراری یا نامعتبر جریمه‌ای ندارند.
* **قابلیت راهنما (With Help):** اگر بازی در حالت راهنما (`with_help=True`) اجرا شود، کاربر می‌تواند با وارد کردن کاراکتر `!` یکی از حروف مخفی کلمه را به صورت تصادفی و با هزینه کردن ۳ حدس آشکار کند.
* **سیستم امتیازدهی:** محاسبه امتیاز نهایی کاربر در صورت برد بر اساس تعداد حدس‌های باقی‌مانده، طول کلمه و تعداد حروف منحصربه‌فرد آن.

### فایل‌ها:

* `hangman.py`: فایل اصلی بازی شامل تمام توابع منطقی، سیستم تعاملی کاربر و حلقه‌ی اصلی بازی
* `words.txt`: بانک اطلاعاتی متنی شامل ۵۵,۹۰۰ کلمه برای انتخاب تصادفی کلمه مخفی توسط کامپیوتر

## 🚀 نحوه اجرا

برای اجرای هر کدام از برنامه‌ها، ابتدا مطمئن شوید پایتون روی سیستم شما نصب است، سپس فایل مربوطه را در ترمینال اجرا کنید:

```bash
# اجرای بازی هنگ‌من
python hangman.py

# اجرای برنامه محاسبه نرخ سود پس‌انداز

```
